# jgpt-web

Next.js UI for JGPT (Garnet Grid theme):
- Mode switch (Power BI / D365FO)
- Model dropdown (gemma, phi3, llama3, mistral, llama3.2)
- Chat with citations panel + tool trace
- Conversation history sidebar

Run:
npm install
npm run dev

**Default Login:**
- Username: `admin@jgpt.com`
- Password: `admin`
